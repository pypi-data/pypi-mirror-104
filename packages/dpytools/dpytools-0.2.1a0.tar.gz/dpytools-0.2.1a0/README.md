# dpytools
Toolset to speed up developing discord bots using discord.py

# Features
- Easy to read typehinted code
- Active development

## Main components
- checks 
- commands
- menus
- parsers

# Instalation
Install the latest version of the library with pip.
```
pip install -U dpytools
```
