Quick start
===========

Install the package:

.. code-block:: console

    pip install --user holdingsparser

Upgrade the package:

.. code-block:: console

    pip install --user --pre -U holdingsparser

Search for a filing with the CIK:

.. code-block:: console

    holdingsparser 0001166559
